Tabelas são estruturas de dados com linhas e colunas.

Tabelas são boas por serem facilmente interpretadas.

Importante saber para tabelas

  • Dê preferência para fazer tabelas com o CSS.
    • Tabelas usadas para leiaute reduzem a acessibilidade para usuários com deficiência visual.
  • Tabelas tem muitas tags, assim poluindo um código html.
  • Tabelas não possuem ajuste automático, sendo reféns de seu elemento pai
    • Uma tabela dentro de uma section estará refém de tudo que for atribuído a ela, assim reforçando o fato de não ser conveniente atribuir tabelas ao html.

Criando tabelas

O conteúdo das tabelas são representadas por <table></table>.
O menor conteiner de uma tabela é representado por <td>.
Para uma linha com quatro células:
<td>Hi, I'm your first cell.</td>
<td>I'm your second cell.</td>
<td>I'm your third cell.</td>
<td>I'm your fourth cell.</td>
(Assim elas ficam alinhadas, mas podem acabar fazendo uma segunda linha se for grande demais.)
Para evitar que as células da tabela cresçam e façam uma segunda linha, deve se colocar dentro do elemento <tr>.
Cada tr fornece uma linha, assim sendo, 2 <tr> diferente, fazem duas linhas. Coloque-os dentro de table.
<th> é para o adicionar cabeçalho na tabela.
