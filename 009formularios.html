Formulários

Elemento <form>:
É um container, vide section, para diferentes tipos de elementos de entrada.

  • input
    • É o elemento de entrada, que pode ser exibido de diversas maneiras, dependendo do atributo type. Pode ser number, text, button, ou vários outros. Podem ser verificados aqui
    Esse é um input do tipo texto:
  • label
    • É um elemento que define rótulo, isso é, ele vai dar um rótulo para o que for digitado no input.
  • select
    • Elemento que define uma lista. Usando, dentro de select, elementos option, você adiciona opções nessa lista.
  • textarea
    • Usuário define um campo de entrada de texto. Uma grandee área para digitar.
  • button
    • Cria um botão. Escrevendo na área de conteúdo o usuário escreve o que estará escrito dentro do botão.
  • fieldset
    • É um elemento usado para definir um grupo de dados de formulários. Abaixo estará o uso de fieldset junto com legend.
  • legend
    • É o elemento que cria um título para seu fieldset, dando uma legenda para ele.
    Pokemons de grama:

    Bulbasaur, Treecko, leafeon
  • datalist
    • Dá uma lista predefinida de sugestões. Por exemplo: Na sua caixa de input terão listas de sugestões. Algo como: "escolha seu inicial" e então ao clicar no input para escrever, aparecerá as opções "charmander", "bulbasaur" e "squirtle". É preciso usar o elemento option para poder criar essa lista.
  • output
    • Ao contrário de input, aqui só tem saída. Então é usado para algo como resultado de um cálculo ou mensagem para o cliente.

  • option
    • Cria opções dentro de categorias que dão essa possibilidade, interagindo com outros tipos de entrada, como datalist e select.
