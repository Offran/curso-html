Imagens

<img>:
O elemento img, da imagem, é um elemento aberto:
<img src="" alt="">
Esse elemento incorpora uma imagem na página.
Em src (source, a fonte) o usuário insere o link da imagem para carregar a imagem no site. A imagem também pode ser inserida diretamente do diretório ou subdiretório. Como explicado em hyperlinks.
Em alt (alternative) o usuário coloca um texto descritivo para usuários que não podem ver a imagem devido a bugs ou problemas visuais do cliente. Geralmente nesses casos uma descrição da imagem pode ser aplicada, por exemplo: "A capa do livro: um dragão do lado de um homem".

Observação: O ideal é que o subdiretório de imagens esteja dentro do mesmo diretório da página html. que faria ficar assim:
<img src="images/imagem.jpg"/>
E o ideal é que se faça exatamente isso, pois imagens vindas de sites irão sobrecarregar fazendo seu site buscar imagens em outros dns. Além de ser antiético e ilegal, caso o dono do site não dê permissão.

Questões de direitos autorais

A maioria das imagens possuem direitos autorais, não exiba imagens no seu site a menos que você cumpra algum desses requisitos:

  • Você seja o dono da imagem.
  • Você recebeu permissão explícita e por escrito do proprietário da imagem.
  • Você tem ampla prova de que a imagem é, de fato, de domínio público.

Largura, altura e títulos

Os atributos width e height, para especificar a largura e altura da sua imagem.
Porém o ideal é sempre usar um editor de imagem antes de colocar uma imagen, para evitar que ela fique distorcida.

title, em imagens, serve para trazer informações de títulos ao passar o mouse por cima. Ele é especialmente útil quando não se tem espaço para descrição da imagem.
O problema é que o atributo title não trás semântica.

Para conseguir semântica é usado <figcaption>, elemento que informa aos navegadores e à tecnologia de assistência que a legenda descreve outro conteúdo do elemento <figure>.

Resumindo:
"alt" fornece informação quando a imagem está ausente ou informação para deficiência visual.
"title" fornece descrição da imagem ao passar o mouse por cima.
"figure" fornece um elemento autocontido, ao qual é o ideal para ser usado com figcaption. Deve ser figure>img>figurecaption
"figcaption" fornece semântica para a imagem.

Imagens de fundo CSS

Também pode colocar imagens no fundo com css:
{
background-image: url("images/imagen.jpg")
}

Imagens colocadas com css são mais fáceis de se posicionar e controlar do que as imagens em html, então é a melhor forma de trabalhar com elas.
Porém não tem semântica, então se a imagem for puramente decorativa, opte por usar css. Também é possível com JS, porém não é o intuito entrar em JS ainda.
