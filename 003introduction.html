Textos

<h1>É um título, nesse caso é o título principal.</h1>
Preferencialmente só se deve usar um H1 por página.
Existem outros títulos menores e de menor importância:

Os mecanismos de pesquisa consideram o conteúdo dos títulos como palavras chaves para encontrar seu site.

<p>É um parágrafo</p>

<span style="font-size:32"></span>
Este não é um título de nível superior.
O elemento span não t em semântica. Isso significa que deve ser usado quando se é pretendido aplicar o CSS ou JS sem atribuir nenhum significado extra. O exemplo dado aqui é de um span que pareça um título h1, porém para mecanismos de buscas e a semântica do HTML, ele não se equipara a qualquer outro título, mesmo sendo aparentemente igual.

Listas:

Existem dois tipos de listas, as não ordenadas e as ordenadas, respectivamente onde a ordem importa e onde não importa.
Não ordenadas: <ul> Ordenadas:<ol>
Cada item dentro da lista é posto dentro de um <li> (item de lista)
Exemplo de uma lista sem importância e prioridade:

  • item X
  • item Y
Exemplo de uma lista com importância e prioridade:
  1. item 1
  2. item 2
Também não há problemas em aninhar uma lista dentro da outra, por exemplo, uma lista sem ordem ter uma observação que possui ordenamento.

Ênfase

Para dar ênfase podemos:
Adicionar itálico com: <i>
Adicionar negrito com: <strong>
Adicionar sublinhado com: <u>
Ou mesmo usar <em>, que denota ênfase, onde geralmente os navegadores aplicam itálico, sendo variável. O ideal nesse caso é usar um span com algum elemento css editável.
O ideal é sempre considerar usar o CSS.
