HTML: Tutorial básico

- Html é uma linguagem de marcação, devido ao foco de estruturar o conteúdo.

- Html é composto por uma série de elementos para criar a estrutura.

Sobre Elementos

- Elementos são itens que contêm dados para o usuário poder ajustar sua estrutura. Eles são utilizados para informar ao navegador que tipo de estrutura está sendo construída. O elemento é a tag de abertura, a de fechamento e o conteúdo. Por exemplo: um elemento pode incluir um bloco de texto, imagem, parágrafo, links, etc.

- Também é possível colocar elementos dentro de elementos. Como na frase:

Meu gato é muito mal humorado.

Que foi escrita da seguinte forma:
<p>Meu gato é <strong>muito<strong/> mal humorado.</p>

- Para o uso preciso de um elemento é necessário informar corretamente os atributos. Caso os atributos não sejam informados o valor padrão do elemento inserido é usado. Em casos onde o elemento é obrigatoriamente vazio o fechamento da tag não é requerida. Exemplo: <br>

Sobre tags e atributos

- Uma tag é iniciada ao abrir os símbolos de "maior que" e "menor que".

- Atributos modificam o comportamento padrão da tag.

Anatomia de um documento HTML

<!DOCTYPE html>:
Atualmente esse elemento está aqui para garantir que tudo do html ocorra corretamente, livre de bugs na sintaxe por parte do computador.

<html>:
Elemento que envolve todo o html, servindo como a base.

<head>:
É como se fosse uma parte por trás do html. Ele inclui detalhes como palavras-chave que aparecerão em resultados de busca, pode receber a parte da estilização (css), entre outras coisas não pertinentes para o tutorial básico.

<meta charset="utf=8">
Define que o conjunto de caracteres será o UTF-8.
O UTF-8 é um elemento que inclui quase todos os idiomas escritos, sendo o mais comum da internet.
O padrão UTF-8 impede que, ao serem separados os bytes de informação, não sejam recombinados.

<title>:
O elemento title define o título da página.

<body>:
O elemento Body contém todo o conteúdo que irá aparecer na página.

Elemento img

<img>:
O elemento img, da imagem, é um elemento aberto:
<img src="" alt="">
Esse elemento incorpora uma imagem na página.
Em src (source, a fonte) o usuário insere o link da imagem para carregar a imagem no site.
Em alt (alternative) o usuário coloca um texto descritiivo para usuários que não podem ver a imagem devido a bugs ou problemas visuais do cliente. Geralmente nesses casos uma descrição da imagem pode ser aplicada, por exemplo: "A capa do livro: um dragão do lado de um homem".

Marcação de texto

Títulos:
Assim como um livro ou apresentação de trabalho, existem títulos mais importantes, de mais alto e mais baixo nível.

Nota: O cabeçaçho <h1> já possui estilo. Não deve se atribuir elementos para aumentar ou deixar em negrito, pois eles são usados para acessibiliade e isso pode desconstruir a ordem semântica da sua estrutura html.

Parágrafos:
<p>: São usados para criarem novos parágrafos.

Listas:
Existem dois tipos de listas, as não ordenadas e as ordenadas, respectivamente onde a ordem importa e onde não importa.
Não ordenadas: <ul> Ordenadas:<ol>
Cada item dentro da lista é posto dentro de um <li> (item de lista)
Exemplo de uma lista sem importância e prioridade:

  • item X
  • item Y
Exemplo de uma lista com importância e prioridade:
  1. item 1
  2. item 2

links:
Elemento <a> (âncora). Exemplo de uso: <a href="link"> Frase ou palavra qualquer que servirá como o link da referência</a>
